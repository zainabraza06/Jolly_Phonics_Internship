What is JollyPhonics?

JollyPhonics is a premium, real-time feedback tool for young learners. By capturing both audio articulation (phonemes) and hand/face gestures (kinesthetic cues), it calculates exact feedback metrics comparing child attempts against validated target templates.

Why Audio + Gestures?

Phonics instruction for children relies heavily on physical gestures (e.g., placing hands to cheeks for 'ai'). Combining these modalities increases phoneme classification accuracy and guarantees a holistic learning environment.

The core technologies powering JollyPhonics

By shifting computational load dynamically, we support instant web analysis without heavy container initialization costs.

🎙️

Whisper Encoder

Utilizes a softmax-weighted sum across layers [1, 3, 5, 7, 9] of the Whisper-small encoder, isolating acoustic properties directly from raw audio waveforms.

✨

MediaPipe Vision

Extracts 234-dimensional joint arrays representing mouth boundaries, poses, and hands, enabling fine-grained posture tracking.

⚡

Cross-Attention Fusion

Fuses sequence modalities at the latent layer via Cross-Attention layers, mapping sound parameters directly to physical actions.

Enterprise AWS Cloud Architecture

JollyPhonics runs on a fully managed AWS infrastructure stack. The backend API is containerized and deployed on AWS Elastic Beanstalk, model assets stream in from Hugging Face Hub on first boot, and static frontends are served globally via CloudFront CDN.
